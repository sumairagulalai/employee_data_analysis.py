# 🎓 Student Result Analysis Using Pandas

A beginner-friendly Python Pandas project for analyzing student marks and generating results.

## 📌 Features

- Read student data from a CSV file
- Calculate average marks
- Generate Pass/Fail results
- Display student performance
- Save processed data to a new CSV file

## 🛠️ Technologies Used

- Python
- Pandas

## 📊 Project Workflow

1. Load student data from CSV
2. Calculate average marks
3. Create a result column
4. Mark students as Pass or Fail
5. Export the final report

## 📁 Output

The processed data is saved as:

result.csv

## 🚀 Learning Outcomes

- Reading CSV files with Pandas
- Working with DataFrames
- Creating new columns
- Using conditional filtering with `loc[]`
- Exporting data to CSV files
